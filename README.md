# c100_door_mount

Remix of ExpiredTapWater's [Tapo C100/C110 Thin Case & Mounting Solution](https://makerworld.com/en/models/137346-tapo-c100-c110-thin-case-mounting-solution#profileId-149453) for direct mounting to the door of my P1S printers.

**Microphone and IR emitters are not retained in this design**

**I don't know if C110 compatability is retained.**

### Part Prep
I printed my parts in black PETG which ends up very shiny. This resulted in some unsightly refections on the glass. Applying fuzzy skin to the inside and painting with flat black paint helped.

### Assembly
1) Remove PCA by prying open the glossy black plastic.
2) Remove Speaker wires
3) Install PCA in the mount bottom, if it rocks, rotate 90°
4) Install qty.4 M2 Heat Set inserts into cover
5) Install cover onto bottom with qty.4 M2X12 Screws

### Installation
1) Use your favorite double sided adhesive tape
2) Stick the camera to the upper left of the door aligned to the black mask. 
	* Powering the camera and watching the video feed is probably a good idea to make sure the tab that blocks the chamber light is in the right spot.
